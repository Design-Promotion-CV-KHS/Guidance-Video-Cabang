Penamaan Folder Resource

Standarisasi pengiriman video agar proses editing lebih cepat dan terstruktur.

Standar File Video

  • Format File Gunakan format .MP4 atau .MOV (H.264) untuk keseimbangan kualitas dan ukuran.
  • Resolusi Minimal Minimal Full HD (1080p). Gunakan 4K jika memungkinkan untuk hasil profesional.
  • Aspek Rasio 16:9 (Landscape) untuk YouTube/Presentasi. 9:16 (Portrait) untuk TikTok/Reels/Shorts.
